# slitherlink-app

- slitherlinkのアプリ開発用
## 概要  (スリザーリンク・パズル作成＆共有アプリ)
　C++で実装したスリザーリンク解法アルゴリズムをコアに、パズルの自動生成・エディタ・オンライン共有機能を備えたクロスプラットフォーム（iOS/Android）対応のパズルアプリ開発プロジェクト
## アーキテクチャ・技術スタック
 * **フロントエンド:** Flutter (Dart)
   * iOS/AndroidのクロスプラットフォームUI構築。
 * **ロジック:** C++ (Native C++)
   * 独自の解法・生成アルゴリズムの実行。
   * Flutterとは dart:ffi を介して連携する。
 * **バックエンド:** Firebase
   * Firestore によるパズルデータの保存・共有、Firebase Auth によるユーザー認証を行う。
## 主要機能（予定）
### 1. パズル自動生成機能
 * C++ソルバーを利用し、指定サイズの「一意解を持つ」パズルをランダムに自動生成する。
### 2. パズルエディタ機能
 * 自動生成された盤面をベースとした編集機能。
 * 空白の盤面に数字を配置する、ゼロからの新規作成機能。
 * 編集操作に伴う、一意解の有無および破綻状態のリアルタイム判定機能。
### 3. オンライン共有＆プレイ機能
 * 作成したパズルをサーバーへアップロードし、公開する機能。
 * 他ユーザーが投稿したパズルを取得・プレイする機能。
## 開発ロードマップ
 * **Phase 1: コアアルゴリズムの構築（完了）**
   * C++によるスリザーリンク・ソルバーの実装および単体テスト。
   * https://github.com/S-KOA/slitherlink-solver
 * **Phase 2: UI構築およびネイティブ連携**
   * Flutterによる基本プレイ画面および盤面UIの実装。
   * dart:ffiを用いたFlutterとC++ソルバーの結合。
   * ローカル環境における「自動生成からプレイ」までの完結。
 * **Phase 3: エディタ機能の実装**
   * 盤面の数字配置・編集用UIの実装。
   * リアルタイム一意解判定・破綻チェックロジックの統合。
 * **Phase 4: オンライン機能の統合**
   * Firebaseの導入（ユーザー認証、Firestore設計）。
   * パズルのアップロード、および一覧取得・プレイ機能の実装。
